# 代码目标

本项目旨在实现一个 mac 上本地的 markdown 文件编辑器，需要实现的功能包括：

- markdown 文件源代码模式的打开和编辑
- markdown 文件的预览，要求支持常见的所有markdown语法（包括但不限于粗体/斜体/删除线/列表/标题行/代码块/表格/图片等，先不需要支持 mermaid）
- markdown 文件预览模式下的编辑能力，主要包括：
  - 文字内容的修改（增加和删除）
  - 各种 markdown 支持格式的修改（包括但不限于粗体/斜体/删除线/列表/标题行/代码块等）
  - 表格的修改（行列的插入/删除，表格item的对齐）
  - 图片的修改（插入，删除，允许用户通过 cmd+c 快捷键来直接插入图片，也允许通过路径插入图片）
  - 允许 cmd+左键 访问超链接，通过系统默认自带浏览器

## 技术栈

仅在 mac 上使用，因此使用 swift c 来开发即可，由于仅在本地进行文件预览和编辑，因此不涉及敏感信息以及存储，不需要后端和数据库
